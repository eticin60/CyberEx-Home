# Icon Dosyaları

Bu klasöre aşağıdaki boyutlarda icon dosyaları ekleyin:

- `icon16.png` - 16x16 piksel
- `icon32.png` - 32x32 piksel  
- `icon48.png` - 48x48 piksel
- `icon128.png` - 128x128 piksel

Icon'lar CyberEx Wallet logosunu içermelidir (neon mavi renk teması).

Icon'ları oluşturmak için:
1. Logo PNG dosyanızı kullanarak farklı boyutlarda export edin
2. Veya online icon generator kullanın
3. Dosyaları bu klasöre kopyalayın
